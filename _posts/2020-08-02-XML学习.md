## XML

1. XML:可扩展标记语言

2. XML用来存储数据
   + 配置文件
   + 在网络中传输

3. XML语法
   + XML文件后缀名为.xml
   + XML文件首行定义文档声明
   + XML文件中有且只有一个根标签
   + XML文件标签区分大小写

4. XML文档组成
   + 文档说明
      1. 属性列表
         + version: 版本号
         + encoding: 编码格式，告知解析引擎当前文档使用的字符集或编码方式
         + standalone: 是否独立，用来决定当前文件是否独立（了解）
   + 指令
   + 标签
     1. 标签不能以数字或标点符号开头
     2. 标签不能包含空格
   + 属性
     + id属性值唯一
   + 文档内容
     + CDATA区：该区域中的数据会被原样展示

     ```xml
        <![CDATA[
        ]]>
     ```
