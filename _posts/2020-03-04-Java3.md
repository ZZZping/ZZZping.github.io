# 方法
   + 格式
   ```
        修饰符 返回值类型 方法名(参数列表) {
            代码 ...
            return ;
        }
   ```
   + 方法名称的命名规则与变量一样
   + Tips
      1. 方法定义的先后顺序无所谓
      2. 方法的定义不能产生**嵌套**、**包含**关系
      3. 方法定义好了之后，不会自己执行，在main函数中需要**调用**
         + 调用方法：方法名称();
