# 数组

## 数组的特点
   1. 数组是一种引用数据类型
   2. 数组中的数据类型统一
   3. 数组长度在程序运行时不可改变

## 数组初始化格式
   1. 动态初始化（指定长度）
      + 
      ```
        数据类型[] 数组名称 = new 数据类型[数组长度]
      ```
   2. 静态初始化（指定内容）
      + 不指定数据个数，直接将具体内容指定
      ```
        数据类型[] 数组名称 = new 数据类型[]{元素1, 元素2, 元素3, ...}
      ```
      + 省略格式：
      ```
        数据类型 数组名称[] = {元素1, 元素2, 元素3, ...}
      ```
