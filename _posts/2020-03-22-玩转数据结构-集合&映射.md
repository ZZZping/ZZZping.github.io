# 集合
   + **Set&lt;E&gt;**
      + void add(E)//不能添加重复元素（无序性）
      + void remove(E)
      + boolean contains(E)
      + int getSize(E)
      + boolean isEmpty()

# 映射
   + 存储（键、值）数据对的数据结构(key,value)
   + 根据键(key)，寻找值(value)
   + **Map&lt;K, V&gt;**
      + void add(K, V)
      + V remove(K)
      + boolean contains(K)
      + V get(K)
      + void set(K, V)//修改K处V的值
      + int getSize()
      + boolean isEmpty()
