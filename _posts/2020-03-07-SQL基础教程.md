# SQL基础教程练习题2
   + 2.1编写一条SQL语句，从product表中选取出 **“登记日期(regist_date)在2009年4月28日之后”** 的商品。查询结果要包含 **product_name** 和 **regist_date**.
   ```
        SELECT product_name, regist_date
        FROM product
        WHERE regist_date > '2009-4-28';
   ```
