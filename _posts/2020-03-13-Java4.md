# 成员变量和局部变量的区别
   1. **定义位置不一样**
      + 局部变量：在方法内部
      + 成员变量：在方法外部，直接下而在类中
   2. **作用范围不一样**
      + 局部变量：只有方法中可以使用，出了方法就不能使用
      + 成员变量：整个类全部可以使用
   3. **默认值不一样**
      + 局部变量：没有默认值，如果使用，需要手动复制
         + 参数在方法调用时，必然会被赋值
      + 成员变量：有默认值
   4. 内存位置不一样
      + 局部变量：位于stack中
      + 成员变量：位于heap中
   5. 生命周期不一样
      + 局部变量：随着方法进栈二出现，随着方法出栈而消失
      + 成员变量：随着对象创建而出现，随着对象被垃圾回收而消失
