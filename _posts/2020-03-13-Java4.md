# 面向对象
   + 面向过程：
   + 面向对象：
      + 封装、继承、多态
      + 类：一组相关属性和行为的集合，一类事物的模板使用事物的属性和行为特征来描述该类事物
         + 属性：事物的状态信息
         + 行为：该事务可以做什么
      + 对象：一类事物的具体体现，对象是类的一个实例，具备该事物的属性和行为
         + 类与对象的关系：
            1. 类是一对事物的描述，是抽象的
            2. 对象是一类事物的实例，是具体的
            3. **类是对象的模板，对象是类的实体**
