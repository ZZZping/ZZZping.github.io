# 关键字
## 关键字特点：
   + 完全小写的字母
   + 在编辑器中有特殊颜色

# 标识符
   + 类、方法等名字为标识符

## 标识符命名规则
   + 包含英文字母，数字，**$**符号和**_**
   + 标识符不能以数字开头
   + 标识符不能是关键字

## 标识符命名规范
   + 类名规范：首字母大写，后面每个单词首字母大写**大驼峰**
   + 变量名称、方法名称：首字母小写，后面每个单词首字母大写**小驼峰**
