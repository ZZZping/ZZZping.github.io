# **時運**
邁邁時運，穆穆良朝。襲我春服，薄言东郊。山涤余霭，宇暧微霄。有风自南，翼彼新苗。<br>
洋洋平津，乃漱乃濯。邈邈遐景，载欣载瞩。人亦有言，称心易足。挥兹一觞，陶然自乐。<br>
延目中流，悠想清沂。童冠齐业，閒咏以归。我爱其静，寤寐交挥。但恨殊世，邈不可追。<br>
斯晨斯夕，言息其廬。花藥分列，林竹翳如。清琴横床，浊酒半壶。黄唐莫逮，慨独在余。
